# Строки


## Определение строк

Вы можете определить строку с помощью цитат: **"**

Например,

```jsx
"кошка"
```
будет строкой, пока

```jsx
кошка
```
является объектом


## Сочетание строк с числом

При добавлении строки к числу (или наоборот), конечный результат будет строкой. Номера имплицитно преобразуются в строки.
