# Variables
## What is a variable?
A variable is a value stored with in the code that can be called back for later usage.
## Defining Variables

You can define variables by using var keyword

For example,

```jsx
var x = 3;
```
will define a variable that is a integer, while

```jsx
var x = "3";
```
will define a string
