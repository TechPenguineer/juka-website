# 职能

## 正在执行函数
每个脚本都应该有一个主函数。 可以通过调用主要函数执行脚本

# 用法示例

```jsx
func x() = { 
    printLine("Hello"); 
}
func main() ={
    x();
}
```

所有函数都用词函数定义。 函数可以包含字母数字值(包括_)，函数不能以数字开头。 职能的定义简单：

```jsx
func foo() =
   printLine("Hello Friend!");
}
func main() = Pow
    foo();
}
```
